
# Lotto Number Generator

## Overview

A simple web application to generate lottery numbers. It features a clean interface, a button to generate new numbers, and a light/dark mode toggle.

## Project Outline

### Style and Design

*   **Layout:** Centered content with a clean and modern look.
*   **Color Palette:**
    *   **Light Mode:** Light background, dark text.
    *   **Dark Mode:** Dark background, light text.
*   **Typography:** Clear and readable fonts. Using 'Poppins' from Google Fonts.
*   **Interactivity:** Buttons with hover effects and animations.
*   **Visual Polish:** Includes a subtle background texture, 3D-style number balls, and button glow effects.

### Features

*   **Lotto Number Generation:**
    *   Generates 6 unique random numbers between 1 and 45.
    *   Displays the generated numbers with a pop-in animation.
*   **Light/Dark Mode:**
    *   A toggle switch to change the color scheme of the application.
    *   The user's preference is saved in local storage.
